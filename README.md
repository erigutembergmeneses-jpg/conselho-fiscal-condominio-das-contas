# 📘 Conselho Fiscal: Com Domínio das Contas

## 🎯 Sobre o Projeto

Site oficial de recursos interativos para o livro **"Conselho Fiscal: Com Domínio das Contas"** – uma plataforma que reúne todos os QR Codes e materiais complementares da obra em um único lugar.

### ✨ Funcionalidades

- **QR Codes Dinâmicos**: Cada recurso possui um QR Code funcional que direciona para conteúdos exclusivos
- **Links Diretos**: Botões de acesso rápido para todos os materiais
- **Design Responsivo**: Adaptado para desktop, tablet e mobile
- **Recursos Exclusivos**:
  - 📹 Vídeo de apresentação do autor
  - 📋 Checklist digital demo
  - 📄 Modelos editáveis (Word/Excel)
  - 🧮 Calculadora de quóruns assembleares
  - ⚖️ Simulador de responsabilidade civil
  - 📊 Dashboard ESG
  - 💡 Calculadora de ROI energético
  - 🗺️ Mapa de riscos interativo
  - 💬 Fórum exclusivo para leitores
  - 📜 Atualizações legislativas em tempo real

## 🛠️ Tecnologias Utilizadas

- HTML5 semântico
- CSS3 com Grid e Flexbox
- JavaScript ES6+
- QRCode.js library
- Font Awesome 6
- Google Fonts (Inter e Poppins)
